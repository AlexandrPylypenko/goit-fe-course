Перечень строчных и блочных тегов

Cписок имен строчных элементов

  • a- созданиe ссылок
  • abbr -указывает, что последовательность символов является аббревиатурой
  • area - определяет активные области изображения, которые являются ссылками
  • b - eстанавливает жирное начертание шрифта
  • bdi - указывает фрагмент текста, который должен быть изолирован от изменения направления вывода текста
  • bdo - устанавливает направление вывода текста
  • br - устанавливает перевод строки в том месте, где этот тег находится
  • abutton - создает на веб-странице кнопки
  • cite - помечает текст как цитату или сноску на другой материал
  • code - предназначен для отображения одной или нескольких строк текста, который представляет собой программный код
  • col - задает ширину и другие характеристики одной или нескольких колонок таблицы
  • data - показывает, что вложенный в него текст является данными такими как, например, величины измерения, денежные суммы в разных валютах
  • dd - задает определение термина, указанного в теге dt контейнера dl
  • del - используется для выделения текста, который был удален в новой версии документа
  • dfn - применяется для выделения терминов при их первом появлении в тексте
  • em - предназначен для акцентирования текста
  • i - Устанавливает курсивное начертание шрифта
  • img - предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG
  • input - предназначен для создания текстовых полей, различных кнопок, переключателей и флажков
  • ins - предназначен для выделения текста, который был добавлен в новую версию документа
  • kbd - используется для обозначения текста, который набирается на клавиатуре или для названия клавиш
  • label - устанавливает связь между определенной меткой, в качестве которой обычно выступает текст, и элементом формы (input, select, textarea)
  • mark - помечает текст как выделенный
  • output - Определяет область в которую выводится информация, преимущественно с помощью скриптов
  • q - используется для выделения в тексте цитат
  • rp - используется для вывода текста в браузерах, которые не поддерживают тег ruby
  • rt - добавляет аннотацию сверху или снизу от текста, заключенного в контейнер ruby
  • rtc - обозначает текстовый контейнер внутри ruby. В основном применяется в качестве описательной части или аннотации для иероглифов
  • s - отображает текст как перечеркнутый
  • samp - используется для отображения текста, который является результатом вывода компьютерной программы или скрипта
  • small - уменьшает размер шрифта на единицу по сравнению с обычным текстом
  • span - предназначен для определения строчных элементов документа
  • strong - предназначен для акцентирования текста. Браузеры отображают такой текст жирным начертанием
  • sub - отображает шрифт в виде нижнего индекса
  • sup - отображает шрифт в виде верхнего индекса
  • textarea - представляет собой элемент формы для создания области, в которую можно вводить несколько строк текста
  • time - Помечает текст внутри тега time как дата, время или оба значения
  • u - добавляет подчеркивание к тексту
  • var - используется для выделения переменных компьютерных программ
  • wbr - указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента

Cписок имен блочных элементов

  • address- предназначен для хранения информации об авторе
  • article -задает содержание сайта вроде новости, статьи, записи блога, форума или др.
  • aside - определяет блок сбоку от контента для размещения рубрик, ссылок на архив, меток и другой информации. Такой блок, как правило, называется «сайдбар» или «боковая панель»
  • audio - добавляет, воспроизводит и управляет настройками аудиозаписи на веб-странице
  • blockquote - предназначен для выделения длинных цитат внутри документа
  • body - предназначен для хранения содержания веб-страницы (контента), отображаемого в окне браузера
  • canvas - создает область, в которой при помощи JavaScript можно рисовать разные объекты, выводить изображения, трансформировать их и менять свойства
  • caption - предназначен для создания заголовка к таблице и может размещаться только внутри контейнера table
  • colgroup - предназначен для задания ширины и стиля одной или нескольких колонок таблицы
  • datalist - создает список вариантов, которые можно выбирать при наборе в текстовом поле
  • details - используется для хранения информации, которую можно скрыть или показать по требованию пользователя
  • div - является блочным элементом и предназначен для выделения фрагмента документа с целью изменения вида содержимого
  • dl - входит в тройку элементов dl, dt, dd, предназначенных для создания списка определений. Каждый такой список начинается с контейнера dl, куда входит тег dt создающий термин и тег dd задающий определение этого термина
  • dt - см. тег dl
  • fieldset - предназначен для группирования элементов формы
  • figcaption - Содержит описание для тега figure
  • figure - спользуется для группирования любых элементов, например, изображений и подписей к ним
  • footer - задаёт «подвал» сайта или раздела
  • form - устанавливает форму на веб-странице
  • h1-h6 - заголовки разных уровней
  • header - задает «шапку» сайта или раздела
  • iframe - создает плавающий фрейм, который находится внутри обычного документа, он позволяет загружать в область заданных размеров любые другие независимые документы
  • legend - применяется для создания заголовка группы элементов формы, которая определяется с помощью тега fieldset
  • li - определяет отдельный элемент списка
  • main - предназначен для основного содержимого документа
  • map - служит контейнером для элементов area, которые определяют активные области для карт-изображений
  • meter - используется для вывода значения в некотором известном диапазоне
  • nav - задает навигацию по сайту
  • noframes - Содержимое тега noframes отображается в браузере, когда он не поддерживает фреймы и не умеет их интерпретировать
  • object - сообщает браузеру, как загружать и отображать объекты, которые исходно браузер не понимает
  • ol - устанавливает нумерованный список
  • optgroup - представляет собой контейнер, внутри которого располагаются теги option объединенные в одну группу
  • option - определяет отдельные пункты списка, создаваемого с помощью контейнера select
  • p - Определяет текстовый абзац
  • pre - определяет блок предварительно форматированного текста
  • progress - используется для отображения прогресса завершенности задачи
  • ruby - предназначен для добавления небольшой аннотации сверху или снизу от заданного текста
  • section - задаёт раздел документа
  • select - позволяет создать элемент интерфейса в виде раскрывающегося списка, а также список с одним или множественным выбором
  • summary - Указывает заголовок для тега details, по которому можно щелкать для разворачивания/сворачивания информации
  • table - служит контейнером для элементов, определяющих содержимое таблицы
  • tbody - предназначен для хранения одной или нескольких строк таблицы
  • td - предназначен для создания одной ячейки таблицы
  • tfoot - предназначен для хранения одной или нескольких строк, которые представлены внизу таблицы
  • th - предназначен для создания одной ячейки таблицы, которая обозначается как заголовочная
  • thead - предназначен для хранения одной или нескольких строк, которые представлены вверху таблицы
  • tr - служит контейнером для создания строки таблицы
  • ul - устанавливает маркированный список
  • video - добавляет, воспроизводит и управляет настройками видеоролика на веб-странице
